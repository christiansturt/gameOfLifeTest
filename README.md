The idea is to implement Conway's game of life. The rules are here:https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life
The skeleton code includes some blank functions for you to fill out, and some example files to test on.
The function print_generation() should print the generation to the console to start with, but for stage 2 I would like it to come up as graphics. There are libraries where you can plot an XY scatter chart. There are even ones that allow you to plot changes over time.
